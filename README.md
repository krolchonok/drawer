# Image Blur & Pixelate Utility (Drawer)

Веб-приложение для быстрого скрытия конфиденциальной информации на изображениях путем размытия (Blur) или пикселизации (Pixelate) выделенных областей.

## Функционал

- **Загрузка изображений**: Перетаскивание файлов или выбор через стандартный диалог.
- **Вставка из буфера обмена**: Поддержка быстрой вставки изображения нажатием `Ctrl+V`.
- **Выделение областей**: Возможность выделить несколько зон на изображении с помощью мыши (ЛКМ).
- **Режимы обработки**:
  - **Blur (Размытие)**: Размывает выделенную область с настраиваемой интенсивностью.
  - **Пикселизация**: Разбивает область на пиксели для скрытия текста или лиц.
- **Интерактивные действия**:
  - Применение эффектов ко всем выделенным областям.
  - Отмена последнего действия (`Ctrl+Z` или кнопка "Отменить закрашивание").
  - Очистка текущих рамок выделения.
- **Экспорт**:
  - Копирование готового изображения в буфер обмена в формате PNG (`Ctrl+C`).
  - Скачивание готового файла.

## Использование

Откройте `index.html` в любом современном браузере. Веб-приложение работает локально и не требует серверной части.
